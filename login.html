HTTP Basic Auth Demo

This is a custom basic auth login page. You can make this page look like anything you like. Much nicer than the ugly built-in browser login dialog right?

Example

Access to this site is restricted to our clients. Since this is just a demo, the username and password are aerobatic.

Invalid credentials

You can intentionally try an incorrect password to see error handling in action. After 5 failed logins within a 5 minute period, a 403 Forbidden error page will be returned which you can also customize with the custom-errors plugin.

To implement basic auth in your own website, see the Aerobatic documentation.
