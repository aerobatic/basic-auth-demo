HTTP Basic Auth Demo

Sometimes you just need to provide a global basic username/password protection on an app. Often HTTP basic authentication is a “good enough” solution. This plugin lets you secure all or part of your app with a username and password stored in environment variables.

We offer two flavors of basic auth: standard and enhanced. With standard basic auth, unauthenticated users are prompted to enter the website username and password in the browser's built-in authorziation dialog. This gets the job done, but it doesn't allow for any customization or branding. Enhanched basic auth, available with all paid plans, allows you to present your website visitors a fully custom login html page.
